Converting Less to Css Demo =========================== Converting `less` files to css files, then use it in html pages. ``` npm install npx lessc index.less index.css npx http-server . -o ``` Open `index.html` in your browser.
package.json 
{
  "dependencies": {
    "less": "^3.8.0"
  },
  "devDependencies": {
    "http-server": "^0.11.1"
  }
}
index.html 
<html>
<head>
    <link rel="stylesheet/less" type="text/css" href="index.css" />
</head>
<body>
<div class="panel">
    <div class="header">Hello</div>
    <div class="body">Less!</div>
</div>
</body>
</html>
index.less 
.panel {
    border: 1px solid #DDD;
    width: 200px;
    height: 300px;
    background-color: #EEE;
    margin: 0 auto;

    .header {
        font-size: 18px;
        font-weight: bold;
        padding: 5px;
    }

    .body {
        font-size: 14px;
        padding: 10px;
    }

}
