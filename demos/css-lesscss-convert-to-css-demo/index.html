Converting Less to Css Demo =========================== Converting `less` files to css files, then use it in html pages. ``` npm install npx lessc index.less index.css npx http-server . -o ``` Open `index.html` in your browser.
index.html
index.html
<html>
<head>
    <link rel="stylesheet/less" type="text/css" href="index.css" />
</head>
<body>
<div class="panel">
    <div class="header">Hello</div>
    <div class="body">Less!</div>
</div>
</body>
</html>
