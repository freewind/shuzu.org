javascript-superagent-logger-demo
JavaScript SuperAgent Logger Demo ================================= ``` npm install node index.js ``` You will see: ``` HTTP GET 200 http://taobao.com/ (356ms) got response ``` This line is logged by the logger: ``` HTTP GET 200 http://taobao.com/ (356ms) ```
index.js
index.js
const request = require('superagent');
const logger = require('superagent-logger');

request.get('http://taobao.com')
  .use(logger)
  .end((err, res) => {
    if(err) return console.error(err);
    console.log('got response');
  });

package.json
{
