Python3 Method Overridden Demo ============================== It's a bit different from Java. You can call any other class's super methods. ``` python3 hello.py ```
hello.py
hello.py
class Parent:
    def hello(self):
        print("Hello from Parent")

class Child(Parent):
    def hello(self):
        print("Hello from Child")

class GrandChild(Child):
    def hello(self):
        print("Hello from GrandChild")
        print("---------- call Child.hello() ---------")
        super(GrandChild, self).hello()
        print("---------- call Parent.hello() ---------")
        super(Child, self).hello()

grand_child = GrandChild()
grand_child.hello()
