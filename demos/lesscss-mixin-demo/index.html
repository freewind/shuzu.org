LessCSS Mixin Demo ================== Much simpler than I thought. ``` npm install npx lessc hello.less hello.css cat hello.css ```
hello.less
hello.less
.center {
  margin: 0 auto;
  width: 400px;
}

.header {
  font-size: 100px;
  .center
}

.body {
  font-size: 16px;
  .center
}
``` $ npx lessc hello.less hello.css $ cat hello.css ``` ``` .center { margin: 0 auto; width: 400px; } .header { font-size: 100px; margin: 0 auto; width: 400px; } .body { font-size: 16px; margin: 0 auto; width: 400px; } ```
