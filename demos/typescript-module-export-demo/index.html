TypeScript Module Export Demo ============================ 怎样把typescript里定义的内容以module的形式export出去？ ``` npm install npx ts-node hello.ts ```
core.ts
core.ts
function hiddenFunction() {
    console.log("I should not be seen from outside");
}

export function helloCore() {
    console.log("Hello from function core");
}

export class Core {
    hello() {
        console.log("Hello from class Core");
    }
}
hello.ts
hello.ts
import * as core from './core';

core.helloCore();

const newCore = new core.Core();
newCore.hello();
