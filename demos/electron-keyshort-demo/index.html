Electron Document Keyshort Demo ========================== It's the same as normal html page to capture keyshorts. ``` npm install npm start ``` Press any key, you will see logs on the window.

index.js
src/index.js
document.onkeyup = function (event) {
    const code = event.keyCode;
    document.getElementById("content").innerText =
        document.getElementById("content").innerText + "You just pressed key code: " + code + "\n";
};
