AngularJS1 Directive Template Url Demo ====================================== Use template from a standalone template file. ``` npm install http-server . -c-1 -o ``` ![demo](./images/demo.jpg) Resources --------- - AngularJS1: https://angularjs.org/ - directive api: <https://docs.angularjs.org/api/ng/service/$compile#directive-definition-object> - $element api: <https://docs.angularjs.org/api/ng/function/angular.element#angularjs-s-jqlite>
Notice: ``` <div my-directive>...</div> ``` Must use `my-directive`, not `myDirective`.
my-directive.html 
<div class="bg-yellow">Good! Template from directive</div>
Notice: ``` app.directive('myDirective', function () { ... }); ``` Must use `myDirective`, not `my-directive`
package.json 
{
  "dependencies": {
    "angular": "^1.7.3"
  },
  "devDependencies": {
    "http-server": "^0.11.1"
  }
}
index.html 
<html>
<head>
    <style>
        .red-border {
            border: 1px solid red;
        }
        .bg-yellow {
            background-color: yellow;
        }
    </style>
</head>
<body ng-app="app">
<div my-directive class="red-border">Hello, world! (should replaced by the directive `myDirective`)</div>
<script src="./node_modules/angular/angular.js"></script>
<script src="./app.js"></script>
</body>
</html>
app.js 
const app = angular.module('app', [])
app.directive('myDirective', function () {
    return {
        templateUrl: './my-directive.html',
        restrict: 'A'
    }
})
