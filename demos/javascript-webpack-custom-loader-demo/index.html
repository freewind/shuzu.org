JavaScript Webpack Custom Loader Demo ===================================== Webpack的loader从概念上讲非常简单，就是一个简单的函数，你把要处理的文本传给它，它会返回处理完的文本。 同时它还可以拿到配置和参数，在这个Demo里未考虑。 ``` npm install npm run demo ``` Then open `index.html`. ![demo](./images/demo.jpg) Resources --------- - Writing a Loader: <https://webpack.js.org/contribute/writing-a-loader/>
entry.js 
const hello = require('./my.hello')

console.log(hello)

hello()
webpack.config.js 
const path = require('path')

module.exports = {
    entry: './entry.js',
    output: {
        path: __dirname,
        filename: 'bundle.js'
    },
    module: {
        rules: [{
            test: /\.hello/,
            use: [
                {loader: path.resolve('./src/hello-loader.js')}
            ]
        }]
    }
}
src/ hello-loader.js 
module.exports = function (source) {
    return `
    module.exports = function() {
        const words = 'Hello, ${source} (from hello-loader.js)!'
        const node = document.createTextNode(words)
        document.body.appendChild(node)
    }
    `
}
