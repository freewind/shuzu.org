chrome-extension-context-menu-selection-text-demo
Chrome Extension Context Menu Demo ================================== Add a menu item to context menu: `Alert selection`. When click on it, the selection content will be alerted. Install: ------- Chrome -> `Extensions` -> Enable `Developer Mode` -> `Load unpacked` -> select the root directory of this project. Open a page, and click the icon of this extension on toolbar, you will a hello alert.
manifest.json
manifest.json
{
  "manifest_version": 2,
  "name": "context-menu-selection-text-demo",
  "description": "context-menu-selection-text-demo",
  "version": "1.0",
  "background": {
    "scripts": ["hello.js"]
  },
  "permissions": [
    "contextMenus"
  ]
}
hello.js
hello.js
function alertSelection(info, tab) {
    alert(info.selectionText);
}

chrome.contextMenus.create({
    title: "Alert selection!",
    contexts: ["selection"],
    onclick: alertSelection
});
