JavaScript Webpack Exclude External Dependencies Demo ===================================================== 在Webpack打包时，可以通过`externals`声明哪些是外部依赖，这样在打包时，会把它们排除在外。 同时，在html页面中，我们需要手动加入这些外部库的引用才行。 ``` npm install npm run demo open index.html ``` ![demo](./images/demo.jpg)
entry.js 
const $ = require('jquery')

$(document).ready(function () {
    $("#main").text("Hello, Webpack (from entry.js)")
})
webpack.config.js 
module.exports = {
    mode: 'development',
    entry: './entry.js',
    output: {
        path: __dirname,
        filename: 'bundle.js'
    },
    externals: {
        jquery: 'jQuery'
    }
}
package.json 
{
  "scripts": {
    "demo": "webpack"
  },
  "dependencies": {
    "css-loader": "^0.23.1",
    "jquery": "^3.3.1",
    "style-loader": "^0.13.1"
  },
  "devDependencies": {
    "webpack": "^4.17.1",
    "webpack-cli": "^3.1.0"
  }
}
index.html 
<html lang="en">
<head>
    <script src="./node_modules/jquery/dist/jquery.js"></script>
    <script src="bundle.js"></script>
</head>
<body>

<div id="main"></div>
</body>
</html>
