AngularJS1 Module Publish To Npm Demo ===================================== 创建了一个module "RedPanel": ``` npm install open index.html ``` ![demo](./images/demo.jpg) 我们把它publish到<https://npmjs.org>上，方便在别的项目中直接引用：<https://www.npmjs.com/package/freewind-angularjs-module-red-panel>
red-panel-module.js
red-panel-module.js
const redPanelModule = angular.module('RedPanel', [])

redPanelModule.directive('redPanel', function () {
    return {
        link: function ($scope, $element) {
            $element.css('border', '1px solid red').css('padding', '10px')
        }
    }
})
