Babel6 Es6 Demo =============== We use old babel6 in this demo(current new version is 7). ``` npm install npm start ``` Will print: ``` Hello, Freewind ``` Reference ========= 1. try babel online: <https://babeljs.io/repl/>
index.js
index.js
class Hello {
	constructor(name) {
		this.name = name;
	}

	hello() {
		console.log("Hello, " + this.name);
	}
}

new Hello("Freewind").hello();
