AngularJS1 Hello World Demo =========================== [AngularJS1](https://angularjs.org/) is not dead yet. ``` npm install open index.html ``` Type your name in the text field, and see the changes. ![demo](./images/demo.jpg)

<html>
<head>
    <script src="./node_modules/angular/angular.js"></script>
</head>
<body>
<div ng-app>
    <input type="text" ng-model="yourName" placeholder="type your name here"/>
    <hr>
    Hello, {{yourName}}!
</div>
</body>
</html>
