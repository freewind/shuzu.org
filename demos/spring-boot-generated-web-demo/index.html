spring-boot-generated-web-demo
Spring Boot Generated Web Demo ============================== This demo is based on the generated code from <http://start.spring.io> (chose `web` in `Dependencies`), and added some code based on the <https://spring.io/guides/gs/spring-boot/>. Run: ``` mvn package java -jar target/demo-0.0.1-SNAPSHOT.jar ``` Visit <http://localhost:8080>, you will see `Hello, spring boot!`
