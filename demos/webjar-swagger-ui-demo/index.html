webjar-swagger-ui-demo
how to run ============ `gradle clean appRun` Then you can visit: `http://localhost:8080/webjars/swagger-ui/2.1.4/` Then you can pass any swagger api description file like `http://petstore.swagger.io/v2/swagger.json` to the input field and view and try it. Note ===== We should use a server container which supports servlet 3.x in order to support webjar directly. # Reference 1. Java project skeleton: http://www.jeroenreijn.com/2012/11/hippo-cms-servlet-3-and-webjars.html
build.gradle
build.gradle
apply plugin: 'java'
apply from: 'gretty.plugin'

gretty {
    // Possible servletContainer values are 'jetty7', 'jetty8', 'jetty9', 'tomcat7', 'tomcat8'. Default is 'jetty9'.
    servletContainer = 'jetty9'
    // httpsEnabled = true
    contextPath = "/"
}

repositories {
    mavenCentral()
}

dependencies {
    compile(
            'io.swagger:swagger-jersey2-jaxrs:1.5.0',
            'org.webjars:swagger-ui:2.1.4'
    )
}
