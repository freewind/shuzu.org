Render LessCSS in Browser Demo ============================== Render `less` files in browser, without converting it to css. ``` npm install npx http-server . -o ``` It will open <http://127.0.0.1:8081> in your browser automatically.
package.json 
{
  "dependencies": {
    "less": "^3.8.0"
  },
  "devDependencies": {
    "http-server": "^0.11.1"
  }
}
index.html 
<html>
<head>
    <link rel="stylesheet/less" type="text/css" href="index.less" />
    <script src="./node_modules/less/dist/less.js" ></script>
</head>
<body>
<div class="panel">
    <div class="header">Hello</div>
    <div class="body">Less!</div>
</div>
</body>
</html>
index.less 
.panel {
    border: 1px solid #DDD;
    width: 200px;
    height: 300px;
    background-color: #EEE;
    margin: 0 auto;

    .header {
        font-size: 18px;
        font-weight: bold;
        padding: 5px;
    }

    .body {
        font-size: 14px;
        padding: 10px;
    }

}
