Ant Merge Jars Demo =================== ``` brew install ant ant hello ``` Check `build` directory and see the two generated jars.
build.xml
build.xml
<project>
    <target name="hello">
        <zip destfile="build/direct-children.jar">
            <zipgroupfileset dir="libs" includes="*.jar"/>
        </zip>
        <zip destfile="build/all.jar">
            <zipgroupfileset dir="libs" includes="**/*.jar"/>
        </zip>
    </target>
</project>
