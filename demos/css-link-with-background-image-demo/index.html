Css Link with Background Image Demo =================================== Create a link using background image. Open `index.html` in your browser, it will show a link like this: ![demo](./images/demo.jpg)
index.css 
body {
    text-align:center;
    padding: 100px;
    font-size: 40px;
}

.github-link {
    width: 100px;
    height: 40px;
    display: inline-block;
    background-image: url("images/github.jpg");
    background-repeat: no-repeat;
    background-size: 100px;
}

.github-link:hover {
    background-size: 120px;
}
index.html 
<html>
<head>
    <link rel="stylesheet" href="index.css">
</head>
<body>
<a class="github-link" href="https://github.com" title="Click Me"></a>
</body>
</html>
