Css github-markdown-css Demo ============================ Apply the github markdown style to your html page, with the help of [github-markdown-css](https://github.com/sindresorhus/github-markdown-css) ``` npm install ``` Then open `index.html` in your browser. The content is from the README file of github-markdown-css project: <https://github.com/sindresorhus/github-markdown-css/blob/gh-pages/readme.md>, looks great.
