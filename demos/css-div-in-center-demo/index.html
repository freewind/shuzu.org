Css Div in Horizontal Center Demo ================================= How to place a div in horizontal center with css. The key is the `auto` part of: ``` margin: 0 auto; ``` Open `index.html` in your browser.
index.css
index.css
.panel {
    width: 600px;
    height: 400px;
    background-color: #EEEEEE;
    font-size: 30;
    line-height: 400px;
    text-align: center;
}

.center {
    margin: 0 auto;
}
index.html
index.html
<html>
<head>
    <link rel="stylesheet" href="index.css">
</head>
<body>
<div class="panel center">This panel is in horizontal center</div>
</body>
</html>
