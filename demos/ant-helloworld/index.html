Ant Demo ======== ``` brew install ant ``` Run: ``` ant ``` or ``` ant hello ``` You will see: ``` hello: [echo] Hello ant! BUILD SUCCESSFUL Total time: 0 seconds ``` If you want to see verbose message, use: ``` ant -verbose ```
build.xml
build.xml
<?xml version="1.0" ?>
<project name="ant-demo" default="hello">
	<target name="hello">
		<echo message="Hello ant!" />
	</target>
</project>
