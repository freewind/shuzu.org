Kotlin Fuel Post Json Demo ========================== Post a POST request with JSON body, with [Fuel](https://github.com/kittinunf/Fuel). The `Content-Type: application/json` head is import for Fuel, otherwise the body json will be sent as form data. Run `Hello.kt` in your IDE.

Hello.kt
src/main/kotlin/example/Hello.kt
package example

import com.github.kittinunf.fuel.Fuel

fun main(args: Array<String>) {
    val result = Fuel.post("http://httpbin.org/post")
            .body("""{"foo": "bar"}""")
            .header("Content-Type" to "application/json")
            .responseString()
    println("------------- request ---------------")
    println(result.first)

    println("--------------- response -----------------")
    println(result.second)

    println("--------------- third ------------------")
    println(result.third)
}
