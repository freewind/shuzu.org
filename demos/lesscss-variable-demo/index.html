lesscss-variable-demo
LessCSS Variable Demo ===================== Use `@` to define and reference variables. ``` npm install npx lessc hello.less hello.css cat hello.css ```
hello.less
hello.less
@width: 100px;
@font-color: '#FF0000';
@strange: abc 123 sdf;

.hello {
  width: @width;
  color: @font-color;
  strange: @strange;
}
``` $ npx lessc hello.less hello.css $ cat hello.css ``` ``` .hello { width: 100px; color: '#FF0000'; strange: abc 123 sdf; } ```
