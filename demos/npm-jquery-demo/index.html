Npm JQuery Demo =============== Use the `jquery.js` from npm jquery module on the html page. ``` npm install -g http-server npm install ``` Start: ``` http-server ``` Visit <http://localhost:8080/index.html>, you will see `hello jquery` on the page.
index.html
index.html
<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>Npm Jquery Demo</title>
	<script src="./node_modules/jquery/dist/jquery.js"></script>
</head>
<body>
	<div id="content"></div>
	<script>
		$("#content").text('hello jquery');
	</script>
</body>
</html>
