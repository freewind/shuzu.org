Npm JQuery Demo =============== Use the `jquery.js` from npm jquery module on the html page. ``` npm install open index.html ``` You will see "hello jquery" on page.
package.json 
{
  "dependencies": {
    "jquery": "^3.1.0"
  }
}
index.html 
<!DOCTYPE html>
<html lang="en">
<body>
<div id="content"></div>

<script src="./node_modules/jquery/dist/jquery.js"></script>
<script>
    $('#content').text('hello jquery')
</script>
</body>
</html>
