TypeScript HapiJS Server Demo ============================= 使用[HapiJS](https://hapijs.com/)来创建一个NodeJS server. 至于为什么我们要使用HapiJS而不使用Express，将在另一个Demo中说明(TODO)。 ``` npm install npm run demo ``` It will start the server: ``` Server running at: http://localhost:8000 ``` And client can access: <http://localhost:8000/hello> with `GET` to get the answer: ``` hello world ``` ![demo](./images/demo.jpg)
server.ts 
import * as Hapi from 'hapi'
import {Request, ResponseToolkit} from 'hapi'

const server = new Hapi.Server({
    host: 'localhost',
    port: 8000
})

server.route({
    method: 'GET',
    path: '/hello',
    handler: function (request: Request, h: ResponseToolkit) {
        return 'hello world'
    }
});

(async () => {
    await server.start()
    console.log('Server running at:', server.info.uri)
})()
