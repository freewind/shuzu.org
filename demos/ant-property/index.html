ant-property
Ant Property Demo ================= ``` brew install ant ``` Run: ``` ant ``` or ``` ant hello ``` You will see: ``` hello: [echo] Hello Freewind! [echo] You like: Java,Scala [echo] Ant properties: [echo] /Users/freewind/workspace/ant-property [echo] /Users/freewind/workspace/ant-property/build.xml [echo] Apache Ant(TM) version 1.10.1 compiled on February 2 2017 [echo] ant-property-demo [echo] 1.8 BUILD SUCCESSFUL Total time: 0 seconds ```
build.xml
build.xml
<?xml version="1.0" ?>
<project name="ant-property-demo" default="hello">
	<property name="name" value="Freewind" />
	<property file="data.properties" />

	<target name="hello">
		<echo message="Hello ${name}!" />
		<echo message="You like: ${like}" />
		<echo message="Ant properties:" />
		<echo message="  ${basedir}" />
		<echo message="  ${ant.file}" />
		<echo message="  ${ant.version}" />
		<echo message="  ${ant.project.name}" />
		<echo message="  ${ant.java.version}" />
	</target>


</project>
