NPM command line demo ===================== Notice `bin` in `package.json`, the file it points will be installed globally, so that should be a file which can be executed. When developing: ``` npm link ```
package.json 
{
  "name": "npm-command-line-demo",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "bin": "bin/npm-command-line-demo",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "",
  "license": "ISC"
}
bin/ npm-command-line-demo 
#!/usr/bin/env node

console.log('hello from "npm-command-line-demo"');
