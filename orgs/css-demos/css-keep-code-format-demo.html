CSS Keep Code Format Demo ========================= How to keep the content of some tag(e.g. `div`, `code`) with format unchanged. The key is `white-space: pre;`. Open `index.html` in your browser.
index.html
index.html
<html>
<head>
    <style>
        code {
            font-family: monospace;
            white-space: pre;
        }

        #code {
            font-family: monospace;
            white-space: pre;
        }
    </style>
</head>
<body>
<code>
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, world!")
    }
}
</code>
<div id="code">
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, world!")
    }
}
</div>
</body>
</html>
