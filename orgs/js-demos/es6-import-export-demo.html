How to run =========== Show `import` and `export` in es6. ``` npm install -g babel-cli npm install ``` Run: There are many files end with `-run.js`, you can run them like: ``` babel-node 01-run.js ``` to view the results(don't forget the source code). Documents: ----------------- Important: - import syntax: <https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/import> - export syntax: <https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/export>
