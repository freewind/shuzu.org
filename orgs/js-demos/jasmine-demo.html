Jasmine Demo ============ A very simple jasmine demo and how to run it in command line ``` npm install ``` Run test: ``` npm test ``` How to create the `jasmine.json` -------------------------------- ``` jasmine init ``` It will create `spec/support/jasmine.json` with default content.

  });
