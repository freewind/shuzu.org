Kotlin Define a Delegate Property without Base Class Demo ========================================================= The delegate property doesn't need to inherit any base class/interface, all it needs is having such one(or two) methods: ``` operator fun getValue(...) operator fun setValue(...) ``` See `Hello.kt`.

Hello.kt
src/main/kotlin/example/Hello.kt
package example

import kotlin.reflect.KProperty

fun main(args: Array<String>) {
    val user = User()
    user.name = "Delegate"
    println(user.name)
}

class User {
    var name by hello("Kotlin")
}

class hello(initValue: String) {
    private var value = initValue

    operator fun getValue(thisRef: User, property: KProperty<*>): String {
        return "Hello, $value!"
    }

    operator fun setValue(thisRef: User, property: KProperty<*>, newValue: String) {
        value = newValue
    }
}
