Ant Macrodef Demo ================= How to define a task in ant. ``` brew install ant ``` Run: ``` ant ``` or ``` ant hello ``` Use `attribute` and `element` to define custom attribute and inner operations. Notice use `@{}` instead of `${}` to use attributes.
build.xml
build.xml
<?xml version="1.0" ?>
<project name="ant-demo" default="hello">

	<macrodef name="hello">
		<attribute name="to-name" />
		<element name="my-task" optional="yes" />
		<sequential>
			<echo message="Hello, @{to-name}" />
			<my-task />
		</sequential>
	</macrodef>

	<target name="hello">
		<hello to-name="java" />
		<hello to-name="ant">
			<my-task>
				<echo message="ant's task" />
			</my-task>
		</hello>
	</target>

</project>
